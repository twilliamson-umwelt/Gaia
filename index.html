⚡ Run Gaia as a Local Web Server ✕

Running Gaia via a local web server (instead of double-clicking the file) unlocks auto-loading of catalogue.csv and avoids browser security restrictions on local files.

Step 1 — Open a terminal and navigate to the Gaia folder:
cd /path/to/gis-inspector
Step 2 — Start the Python web server:
python -m http.server 8080
Step 3 — Open in your browser:
http://localhost:8080/index.html

Python 3 is pre-installed on most systems. On Windows, try py -m http.server 8080. Keep the terminal open while using Gaia.

Gaia v1.0 — GIS Data Inspector
✕

A portable, browser-based GIS viewer built for the Umwelt team.

Loading Data

Drag files directly onto the map, or use the Load Data panel on the left. Supported formats:

  • Shapefile — drop the .shp, .dbf, .prj together
  • GeoJSON / JSON — standard vector format
  • KML / KMZ — Google Earth files
  • CSV / TXT — auto-detects lat/lng or x_easting/y_northing columns; also reads wkt_geometry for lines and polygons
  • ZIP — drop a zip containing any of the above
  • .gaia — Gaia session export file (restores all layers)

Add Layer from URL

Click the + Add Layer from URL button to connect to:

  • Catalogue — your team's service list (CSV with Group, Name, URL). Drop a catalogue.csv onto the drop zone or browse for it. Loads at current map extent.
  • ArcGIS REST — FeatureServer or MapServer layer URLs
  • WMS / WMTS / XYZ — raster tile services
  • GeoJSON URL — any endpoint returning GeoJSON

Attribute Table

Open the attribute table from the panel or by using ⊟ Open Attribute Table in a layer's context menu (⋯).

  • Resize columns — drag the right edge of any column header
  • Reorder columns — drag the ⠿ handle on the left of a column header
  • Sort — click a column header to sort ascending/descending
  • Expand a cell — double-click any cell to see its full value
  • Copy a value — right-click any cell to copy the full value to clipboard
  • Select by attribute — use the ⊞ button to filter by field value
  • Select features — click map features, or use Ctrl+click / Shift+click for multi-select

Layer Symbology

Click the geometry icon (●/—/⬡) on any layer to open the symbology picker. You can set fill colour, outline colour, and — for point layers — choose between circle, square, and triangle markers.

Exporting Data

  • Export layer — use the ↓ Export panel to download as GeoJSON, KML, CSV, or WKT; optionally reproject to a different CRS first
  • Save Locally — saves your workspace to the browser (restores automatically next time)
  • Export Session — downloads a .gaia file you can share with colleagues or load into another Gaia instance

Other Tools

  • Measure — measure distances and areas via the Widgets button
  • Buffer — create buffer polygons around features
  • Select by Location — spatial selection using a drawn shape
  • Create Features — digitise new point, line, or polygon layers
  • Right-click map — copy coordinates in multiple formats, add a point
  • CRS — change the coordinate display in the bottom bar
  • ⚡ Local Server — instructions for running Gaia via Python's HTTP server, which enables auto-loading of catalogue.csv
